# Data Structures Used:

- Doubly linked list:
1. Allows efficient insertion and deletion of nodes.
2. The presence of both prev and next pointers makes it easier to adjust links when nodes are added or removed.

# Functions Description

- insertAtEnd(int val) = Inserts a new node at the end of the list.
- deleteNode(int val) = Deletes a node containing the specified value.
- display() = Displays all nodes from head to tail.
- main() = Demonstrates insertion and deletion.
